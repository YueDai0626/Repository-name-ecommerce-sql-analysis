# E-commerce Sales SQL Analysis

## Project Overview
This project analyzes e-commerce transaction data using SQL to generate key business insights including revenue trends, customer segmentation, product performance, and regional sales distribution.

The goal is to demonstrate practical data analysis skills used in real-world business analytics environments.

---

## Dataset Description
The dataset contains transactional sales records including:

- Order information
- Customer segments
- Product categories
- Pricing and quantity
- Payment methods
- Geographic regions
- Transaction dates

---

## Database Schema
A relational table was created to store structured sales data including:

- Order ID
- Customer ID
- Segment
- Region
- Product details
- Revenue metrics

See `schema.sql` for table creation.

---

## Business Questions Answered

1. Total revenue generated
2. Revenue by region
3. Top-selling products
4. Revenue by product category
5. Monthly revenue trends
6. Revenue by customer segment
7. Payment method usage patterns

See `analysis.sql` for queries.
